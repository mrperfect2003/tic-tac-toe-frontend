#  Tic Tac Toe Frontend

A real-time multiplayer Tic Tac Toe game frontend built using **Next.js** and integrated with **Nakama** for backend game server functionality.

##  Features
-  Interactive Tic Tac Toe game UI
-  Real-time multiplayer gameplay
-  Integration with Nakama server
-  Matchmaking support
-  Responsive design

##  Tech Stack
- **Next.js** – React framework for frontend
- **JavaScript**
- **CSS**
- **Nakama** – Game server for real-time multiplayer

##  Project Structure
- `components/` – Reusable UI components
- `pages/` – Next.js pages (routing)
- `lib/` – Utility and service functions (API, Nakama client)
- `styles/` – Styling files
